AMAZON PRODUCT EXTRACTOR

Overview
This extension extracts the product details such as product name , product price and product image from the website and displays them on the extension's popup window.


Installation
Download the project files from the provided link.
Extract the files to a local directory on your computer.


Loading the Plugin in Chrome
Open Chrome and go to chrome://extensions/.
Enable Developer Mode by toggling the switch in the top right corner.
Click the Load unpacked button and select the folder containing the plugin files.
The plugin will now appear in your list of installed extensions.


To Test the Plugin 
Go to the product page (https://www.amazon.in/gp/product/B0CX58MTNN/ref=s9_bw_cg_Budget_3a1_w?pf_rd_m=AT95IG9ONZD7S&pf_rd_s=merchandised-search-3&pf_rd_r=77P66SSB310J5T7MSV2T&pf_rd_t=101&pf_rd_p=73871b5a-41ae-4742-ab0c-18601828bb82&pf_rd_i=1389401031)
Click on the plugin icon in the Chrome toolbar.
A popup will appear showing the product's name, price, and image.
